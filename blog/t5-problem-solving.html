I learnt to be patient and be able to laugh at my silly mistakes. After all, it is all part of the learning process, right? I love having the support of my peers in my cohort. I really look forward to meeting them all in person, and working together to solve many more problems!

Explain to your non-technical friend:

  • .map()
  • .fileter()
  • .reduce()

.map()

As the name suggests, this JaveScript function 'maps out' to the items in the original array (list of items), and then carries out a task, a procedure for each item in that list, showing the results in a new array.

.filter()

This function 'checks' the items in the original array and passes them though a 'test'. If the items pass this test successfully, then they're placed in a new array.

.reduce()

I'm not exactly sure about this one. But I think it turns the original function into a smaller value, reading each item in an array from left to right.
